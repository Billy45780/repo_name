ПР №5 — CSS Grid

Какое основное свойство включает CSS Grid и чем отличается от Flexbox?
Grid включается свойством display: grid. В отличие от Flexbox, который работает только в одном направлении (по строке или по столбцу), Grid создаёт полноценную двумерную сетку.

Разница между grid-template-columns и grid-auto-columns:
grid-template-columns задаёт фиксированные колонки сетки, а grid-auto-columns — ширину автоматически добавляемых колонок, если элементов больше, чем определено шаблоном.

repeat(), minmax(), auto-fit:

repeat(3, 1fr) — создаёт 3 одинаковые колонки;

minmax(200px, 1fr) — ограничивает размер между минимумом и максимумом;

auto-fit — автоматически подгоняет количество колонок под ширину контейнера.

Именованные grid-области:
Позволяют давать областям сетки имена и размещать элементы через grid-area, что делает код понятнее.

Нумерация линий и grid-column / grid-row:
Grid-линии нумеруются от 1. Например, grid-column: 1 / 3; — элемент займёт место от линии 1 до 3.

grid-gap (gap):
Добавляет промежутки между ячейками без использования margin, что удобнее и безопаснее при адаптивной верстке.

justify-items и align-items:
Отвечают за выравнивание элементов внутри ячеек по горизонтали и вертикали соответственно.

Разница между auto-fill и auto-fit:
auto-fill сохраняет пустые ячейки, auto-fit — подгоняет сетку, убирая пустые места.

Сложные макеты с grid-template-areas:
Можно нарисовать структуру страницы с помощью текстового шаблона. Имена должны быть уникальны и согласованы в HTML.

Преимущества CSS Grid:
Удобен для создания сложных макетов, гибкий и хорошо работает с адаптивностью.
