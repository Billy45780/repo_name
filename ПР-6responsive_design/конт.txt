+ПР №6 — Адаптивная верстка

Viewport meta tag:
<meta name="viewport" content="width=device-width, initial-scale=1.0"> — делает страницу адаптивной на мобильных устройствах.

Подходы к адаптивности:
Использовал responsive-дизайн с медиа-запросами и гибкими единицами.

Mobile First:
Сначала верстается под мобильные устройства, потом добавляются стили для больших экранов. Это проще и эффективнее.

Единицы измерения:
Использую %, rem, vw/vh, потому что они подстраиваются под размер экрана.

Функция clamp():
Позволяет задать адаптивный размер, например font-size: clamp(1rem, 2vw, 2rem) — изменяется в зависимости от ширины экрана.

Брейкпоинты:
Основные — 1200px, 1024px, 768px, 480px. Определяются под популярные устройства.

Медиа-запросы:
Пример:

@media (max-width: 768px) {
  body { font-size: 14px; }
}


Разница между max-width и min-width:
max-width — стили для экранов меньше заданного размера, min-width — для экранов больше.

Адаптивные изображения:
Используется max-width: 100%; height: auto;, чтобы не выходили за границы контейнера.

Инструменты тестирования:
Использую DevTools в Chrome — режим адаптивности, просмотр под разные устройства.
